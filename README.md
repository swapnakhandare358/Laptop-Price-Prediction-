# 💻 Laptop Price Prediction

## Project Overview

This project predicts laptop prices based on user specifications using Machine Learning. The application is developed with Python and deployed using Streamlit.

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Streamlit

## Features

* Data Cleaning
* Exploratory Data Analysis (EDA)
* Machine Learning Model
* Laptop Price Prediction
* Interactive Streamlit Web Application

## How to Run

1. Clone the repository.
2. Install the required libraries:
   pip install -r requirements.txt
3. Run the application:
   streamlit run app.py
