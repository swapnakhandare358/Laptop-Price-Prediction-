# 💻 Laptop Price Prediction using Machine Learning & Streamlit

## 📖 Overview

This project is a Machine Learning-based web application that predicts the price of a laptop based on its specifications. The application is developed using **Python** and deployed with **Streamlit**, providing an interactive and user-friendly interface for real-time price prediction.

---

## 🚀 Key Features

✔️ Interactive Streamlit Web Application

✔️ Real-Time Laptop Price Prediction

✔️ Clean and Responsive User Interface

✔️ Machine Learning Prediction Model

✔️ Easy-to-Use Input Form

---

## 🛠️ Tech Stack

- Python
- Streamlit
- Pandas
- NumPy
- Scikit-learn
- Pickle

---

## 📊 Input Parameters

The prediction model uses the following laptop specifications:

- Company
- Laptop Type
- RAM (GB)
- Operating System
- CPU
- GPU Brand
- Weight
- Screen PPI
- HDD Storage
- SSD Storage
- Touchscreen
- IPS Display

---

## 💰 Output

🎯 Predicts the estimated laptop price based on the selected specifications.

## ▶️ How to Run

### Clone the Repository

```
git clone https://github.com/yourusername/Laptop-Price-Prediction.git
```

### Install Required Libraries

```
pip install -r requirements.txt
```

### Run the Streamlit App

```
streamlit run app.py
```
